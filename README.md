# AI Powered Sales Forecasting Dashboard

This project predicts future retail sales using historical Superstore data and machine learning techniques.

## Dataset
Sample Superstore Dataset

## Tools Used
- Python
- Pandas
- Matplotlib
- Prophet

## Project Workflow
1. Load and clean the dataset
2. Aggregate daily sales data
3. Visualize sales trends
4. Build forecasting model
5. Predict future sales

## Output
- Sales trend visualization
- 30-day sales forecast graph
